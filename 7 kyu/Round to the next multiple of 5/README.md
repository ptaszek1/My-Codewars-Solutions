<h1>Round to the next multiple of 5.</h1>

<p>Given an integer as input, can you round it to the next (meaning, "higher") 5?</p>
<h2>Examples:</h2>
<ul>
<li>0    ->   0</li>
<li>2    ->   5</li>
<li>3    ->   5</li>
<li>12   ->   15</li>
<li>21   ->   25</li>
<li>30   ->   30</li>
<li>-2   ->   0</li>
</ul>
