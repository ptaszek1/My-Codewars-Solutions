<h1>Grader</h1>

<p>Create a function that takes a number as an argument and returns a grade based on that number.</p>

Anything greater than 1 or less than 0.6 	'F' <br>
0.9 or greater 	"A"  <br>
0.8 or greater 	"B" <br>
0.7 or greater 	"C" <br>
0.6 or greater 	"D" <br>
<h2>Examples</h2>
<ul>
<li>grader(0.9) == 'A'</li>
<li>grader(0.3) == 'F'</li>
<li>grader(234) == 'F'</li>
<li>grader(0.75) == 'C'</li>
</ul>
